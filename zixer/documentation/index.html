Installation

System Requiement
  • Backend Framework: Built on Laravel 6x
  • Frontend Framework: Built on Bootstrap 4x
  • Requires PHP Version >= 7.3
  • Database Support MySQL, Mysqli.
Installation Proccess

01. Download file from codecany and extract in your pc and you will get 03 folder. you will find a folder for database a folder for installable zip file another one is documentation.

02. Now Create a database for this script and upload give database inside database folder.

03. Upload installable file to server and unzip it.

04. After that edit database info in installable file > @core > .env

05. Update below code with your database information and that's it you are good to go.

DB_DATABASE=YOUR_DATABASE_NAME
DB_USERNAME=YOUR_DATABASE_USENAME
DB_PASSWORD=YOUR_DATABASE_PASSWORD
Installation done. Enjoy!!!
